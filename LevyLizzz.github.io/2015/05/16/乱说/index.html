只想着目标的过程是痛苦的，学着享受过程，目标变不变谁都不知道，毕竟活在裆下。

今天又没去图书馆呢…Git今天又有问题了hexo deploy已然发布不上来，提示成功后我在Github/GitCafe里看不到commit。但是我7点手动push静态网页的时候成功刷新发现在下午15:15 17:42: 15:51提交了三次 _(:з」∠✿)_ _(:з」∠)_ 这只扑街的猪在这么好看的字体下不能完美显示是有道理的！！(学习了一波Markdown语言还是显示出来了)所以为什么push成功gitcafe为什么不能立马显示呢。。。没听说别人有延迟的现象。
晚上继续把站内搜索弄好，我用的swiftype,很棒的站内搜索引擎。有时间还是把自己踩的坑记录下吧。希望今晚能早点睡吧，昨天又是四五点睡觉 ﹁_﹂ 两点钟push了第一个commit知乎大法又陪我玩了两个多小时 ﹁_﹂ 睡了五六个小时就没再休息的。。一定要改啊！我不要猝死啊。

晚上看了个天天向上，那个特警大队长一个镜头灭光了所有小鲜肉看得我高潮。所以我应该还是直男癌咯！
其实纯粹想多写几篇文章看看折叠效果，所以乱写的，结尾点题！有人表扬我吗？
